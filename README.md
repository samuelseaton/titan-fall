# Colossus Break

A browser-based action arcade game built with vanilla HTML5 Canvas. No frameworks, no dependencies — just one HTML file, one JS file, one CSS file.

## Gameplay

You play as a lone soldier using a grapple hook to swing through the air and cut down approaching Colossi before they reach you.

- **Hold left-click** near a hook to latch on
- **Move your mouse** to steer mid-swing
- **Release** to fly through the air
- **Press Space** to slash — get close to the glowing eye to land the kill

The world scrolls faster as your score climbs. Don't let a Colossus walk past you.

## Controls

| Input | Action |
|---|---|
| Hold left-click | Grapple to nearest hook |
| Mouse movement | Steer while swinging |
| Release click | Let go and fly |
| Space | Slash attack |

## Built With

- HTML5 Canvas 2D
- Vanilla JavaScript (no libraries)
- CSS3

## Running Locally

Just open `index.html` in a browser. No build step, no server required.
